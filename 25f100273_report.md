## INTRODUCTION

Chest Radiology is a commonly used imaging technique for diagnosing diseases related to lungs and thoracic region. Since it is more challenging and time consuming to perform these diagnosis manually, deep learning makes easier to conduct these diagnosis with better precision in lesser time.

The aim of this project is to develop a deep learning model that can classify X-ray images into several classes such as Atelectasis, Cardiomegaly, Consolidation, etc. including a 'No Finding' class. 
The inputs comprised of four files, namely images which contained the images of the X-ray, training dataset which was used to train the model, test dataset that measured the accuracy of the model and the sample submission which specified the format of the submission file.

The objective of the model was to test and compare varioys CNN based models, such as Custom CNN, ResNet, EfficientNet80 and VGG16, to achieve accurate and efficient classification. 


---


## METHODOLOGY

###  Data Preprocessing 

Each row contains an image id, and binary inidicator for each type of disease. The model requires conversion of multi-label classification into a single-label classification, such that only column is 1 and the resta re 0. All images were loaded and resized to a fixed resolution (128x128 or 96x96 pixels) and images were enhance using the brightness and contrast function.


### Data Loading 

A custom generator was implemented to load images in batches during training, reducing the memory usage and improving efficiency. It also shuffles the data at each epoch. 

  
### Models Used

1. **Custom Convolutional Neural Network**: CNN model is used for image classification and it learns patterns such as edges, shapes, textures.
   * This was chosen first because it was a simple baseline model that is easy to train and fast.
    
 2. **ResNet50 (Residual Network)**: ResNet is used to classify images by solving the problem of vanishing gradients by skipping connections.
    * This was chosen because it is more accurate as compared to CNN and is capable of performing better on complex images.
   
3. **EfficientNetB0**: EfficientNetB0 is used to balance depth, width and resolution by compund scaling.
   * This was chosen to ensure faster training while maintiaing the performance.
  
4. **EfficientNetB3**: EfficientNetB3 is a more powerful version of EfficientNet.
   * This was chosen to able to capture more fine details missed by B0, to improve the feature extraction and achieve higher accuracy.

5. **VGG16**: VGG16 is a neural network consisting of 16 layers. It was used as a pretrained model with transfer learning.
   * It was initially tested using transfer learning. It provided reasonable performance but was computationally heavier.


### Model Training

All models were initially trained using the following parameters:
* Optimizer: Adam
* Learning Rate: 0.0001
* Loss Function: Sparse Categorical Crossentropy
* Batch Size: 16
* Epochs: 10

Transfer Learning was applied for ResNet and EfficientNet Models, where pretrained weights from ImageNet were used. 


### Techniques Used 

1. *Data Augmentation*: Brightness and contrast adjustments were applied to increase data variability and improve generalization.

2. *Transfer Learning*: Pretrained models (ResNet and EfficientNet) were used to use existing feature knowledge

3. *Regularization*: Dropout layers were added to prevent overfitting by randomly deactivating neurons during training.

## OBSERVATIONS

* ResNet50, EfficientNetB0, and EfficientNetB3 achieved identical scores, indicating that increasing model complexity did not significantly improve performance under the given setup.
* VGG16 performed slightly better than CNN but was slower and more computationally expensive.
* EfficientNetB0 (optimized configuration) significantly reduced training time without affecting performance.
* EfficientNetB3, despite being more advanced, did not outperform simpler models due to limitations in the training setup.
* The results suggest that performance is more dependent on training strategy and problem formulation than model architecture alone.

## CONCLUSION


In this project, multiple deep learning models including CNN, VGG16, ResNet50, EfficientNetB0, and EfficientNetB3 were implemented and evaluated for chest X-ray image classification. 
The experimental results showed that increasing model complexity did not significantly improve the final score. Most advanced models, including ResNet50 and EfficientNet variants, achieved similar performance, indicating that the limitation lies not in the model architecture but in the training setup and problem formulation.
Among all models, EfficientNetB0 (optimized configuration) emerged as the most practical choice, as it achieved comparable performance while significantly reducing training time. EfficientNetB3, although more powerful, did not provide additional benefits under the current configuration.
The results highlight the importance of factors such as efficient preprocessing, proper training strategies, and optimization techniques over simply increasing model depth. 
Overall, this study demonstrates that while deep learning models are effective for medical image classification, achieving optimal performance requires a balanced combination of model selection, computational efficiency, and training strategy.
